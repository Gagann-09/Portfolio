## 🌟 About The Project

Welcome to my personal portfolio repository! This project serves as a central hub for my professional identity, showcasing my work in Machine Learning, Full-Stack Development, and System Architecture. 

The application is built on a **"Cyber-Premium"** design system—a blend of deep space darks (`#050505`) and vibrant cyber-orange accents (`#e86c2f`), leveraging cutting-edge web technologies to deliver an immersive, sub-100ms latency experience.

### ✨ Key Features

- **Blazing Fast Performance**: Statically generated with Next.js App Router and optimized using Rust-based Turbopack.
- **Immersive 3D Canvas**: Hardware-accelerated particle background powered by React Three Fiber (R3F).
- **Fluid Micro-Interactions**: Complex, declarative scroll and hover animations powered by Framer Motion.
- **Enterprise-Grade Security**: Strict HTTP headers (CSP, HSTS, X-Frame-Options) implemented globally.
- **Responsive & Accessible**: Fully fluid layout ensuring perfect rendering from mobile devices to ultrawide monitors.

## 🚀 Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

You will need Node.js and npm installed on your machine.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Gagann-09/Portfolio.git
   ```
2. Navigate into the project directory
   ```sh
   cd Portfolio
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the development server (powered by Turbopack for ultra-fast HMR)
   ```sh
   npm run dev
   ```
5. Build for Production
   ```sh
   npm run build
   npm start
   ```

## 🔒 Security Hardening

Security is treated as a first-class citizen. The application uses a robust `next.config.mjs` to enforce strict security headers, including:
- **Content-Security-Policy (CSP)** to mitigate XSS attacks.
- **Strict-Transport-Security (HSTS)** to enforce secure connections.
- **X-Frame-Options** to prevent clickjacking.
- **Permissions-Policy** to lock down browser feature access (camera, microphone, geolocation).
